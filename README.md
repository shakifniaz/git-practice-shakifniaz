# Git Practice Project

Name: Shakif Niaz

## Project Description:
This project was created as part of the Module 8 Assignment for Ostad. It demonstrates the fundamental concepts of version control using Git and repository management with GitHub.

## Features
- Display the student's name
- Display the current date
- Perform addition and subtraction
- Perform multiplication and division
- Handle division-by-zero errors
- Demonstrate Git branching and merging workflows

## Learning Objectives
- Repository creation
- Git initialization
- Commits
- Branching
- Merging
- Pushing code to GitHub
